## Installation 

`npm i dcord`


### Usage 

```
const bot = require('dcord')

bot.on('<YOUR_BOT_TOKEN>')
```


### Customize

You Can Customize The Prefix Settings.
``` 
const bot = require('dcord')

bot.on('<YOUR_BOT_TOKEN> <PREFIX>')
```


### Example 

Here's Some Example
```
const bot = require('dcord')
const token = YOUR_BOT_TOKEN
const prefix = "!"
bot.on(token + " " + prefix)

```

### Commands

  - get <STOCK_QUOTE>
